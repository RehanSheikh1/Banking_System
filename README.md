# 🏦 Ruby Console Banking System

A simple **console-based banking application** built using Ruby.  
This project allows users to create accounts, deposit and withdraw money, transfer funds, check balance, and delete accounts — all from the terminal.

---

## 🚀 Features

- Create a new bank account
- Deposit money into account
- Withdraw money from account
- Transfer money between accounts
- Check account balance
- Delete account
- Transaction history tracking
- Simple menu-driven console interface

---

## 🧠 Concepts Used

- Ruby Classes and Modules
- Arrays and Hashes
- Object-Oriented Programming (OOP)
- Loops and Conditionals
- User Input Handling (`gets.chomp`)
- Basic data management in memory
